# Futur Utils
Общие утилиты для написания JavaFX приложений

|Модуль                     |Описание
|:--------------------------|:-
|ui                         |Вспомогательные классы контролов интерфейса
|common                     |Вспомогательные классы математики, работы со строками, файлами, ресурсами и т.д.
